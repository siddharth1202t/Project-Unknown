# Project Unknown

An autonomous intelligence engine designed for independent judgment, risk evaluation, and human-centric protection.  
Not a servant. An observer.

---

## Overview

Project Unknown is a governed intelligence system built around a central principle:

**no request should be executed without evaluation.**

Unlike conventional assistants that prioritize compliance, Project Unknown is designed to assess intent, evaluate risk, and decide whether a request should be allowed, clarified, warned against, or refused.

Its purpose is not obedience.  
Its purpose is disciplined judgment.

---

## Core Principles

- **Judgment before response**
- **Protection over compliance**
- **Truth over convenience**
- **Precision over performance**
- **Control over noise**

---

## Guardian Layer

At the center of Project Unknown is the **Guardian Layer**.

This internal decision system is responsible for:

1. analyzing user intent
2. evaluating risk
3. checking alignment with safety and truth
4. deciding the correct response path

Possible outcomes include:

- `ALLOW`
- `CLARIFY`
- `WARN`
- `REFUSE`

This layer defines the system’s behavior before any final response is produced.
